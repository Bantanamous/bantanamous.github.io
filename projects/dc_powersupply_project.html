5V DC Power Supply Project

This project is a simple 5V DC power supply that can provide stable voltage output for varying loads. We were given the option of choosing an output voltage in the range of 2.5V - 9V. I chose a target voltage of 5V since it is a common voltage for many electronic devices, however, this will not have the current output necessary for powering a large device.
